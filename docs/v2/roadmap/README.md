# v2 Roadmap Index

## 目的

这个目录只负责记录每条主线当前推进到哪一步。

它不重复 requirement、design、spec 的正文，只保留最小状态信息：

- 当前状态
- 当前焦点
- 已完成产物
- 当前阻塞

## 结构规则

- 一个长期主线对应一个 roadmap 文件
- roadmap 文件只做推进索引，不写实现细节
- 当 requirement、design、spec 或实现有推进时，更新对应 roadmap
- 如果需要看全局状态，回到 `docs/v2/ROADMAP.md`
